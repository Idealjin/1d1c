객체

객체의 선언 및 호출

객체는 키값을 속성을 식별한다.
중괄호({})를 사용해서 객체를 생성하고, 속성에는 모든 자료형이 올 수 있다.

객체의 키 식별자 테스트

객체의 메소드 속성

객체의 속성 중 함수 자료형인 속성을 메소드라고 부른다.

객체와 반복문

객체의 속성을 살펴볼 때에는 단순 for문으로 사용 불가능하고
for in 문을 사용해야 한다.

in과 with키워드

in : 객체 내부에 해당 속성이 있는지 확인하는 키워드
with : 코드를 줄여주는 키워드, 호출 시 객체명 생략이 가능하다.





객체의 속성 추가와 제거

처음 객체 생성 이후 속성을 추가하거나 제거하는 것을 동적으로 속성을 추가한다 혹은 제거한다 라고 한다.
